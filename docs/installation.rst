Installation
============

To install from PyPI, run:

.. code-block:: bash

    pip install timekill

To install from source, run:

.. code-block:: bash

    git clone https://github.com/ErikBjare/timekill
    cd timekill

    pip install -e .
    # or...
    poetry install

Now, just follow the instructions on the :doc:`Usage <usage>` page.
