Usage¶

Here we document basic usage.

To get started, first install the package.

Now, let’s see how it would plan your day, run:

$ timekill plan

A full CLI reference can be found below.

CLI¶

timekill¶

A better way to kill time, a tool for planning your day and finding things to do.

NOTE: still in early development

timekill [OPTIONS] COMMAND [ARGS]...

list¶

List content with recommendations.

timekill list [OPTIONS]

plan¶

Plan a day of activity.

timekill plan [OPTIONS]

Options

--until <until>¶

server¶

Subcommands for running a server (WIP)

timekill server [OPTIONS] COMMAND [ARGS]...
start¶

Entrypoint for the timekill server

timekill server start [OPTIONS]

suggest¶

Suggest activities to do.

timekill suggest [OPTIONS]
